# puruslang/assets

Static assets for the Purus language project.

## Usage

Reference via GitHub raw URL:

```
https://raw.githubusercontent.com/puruslang/assets/main/<path>
```

## Files

| File | Description |
|---|---|
| `logo.png` / `logo.svg` | Purus logo |
| `icon.png` / `icon.svg` | Purus icon |
| `banner.png` | OGP / social share banner |
| `favicon.ico` / `favicon.svg` | Favicon |
| `img/` | Background variants (icon/logo on black/white) |
| `vscode/` | VS Code extension icons and icon theme |
